# Korean Plastic Surgery Campaign Feed

韓国美容整形クリニックのキャンペーン・モニター募集情報をまとめたフィードサイト

## Features

- TOP 10のキャンペーン情報
- X（Twitter）の投稿埋め込み
- レスポンシブデザイン
- カテゴリー別バッジ表示

## Tech Stack

- HTML5
- TailwindCSS (CDN)
- Twitter Widgets API
- Vanilla JavaScript

## Local Development

ブラウザで `index.html` を開くだけで動作します。

## Deployment

Vercel にデプロイ済み
